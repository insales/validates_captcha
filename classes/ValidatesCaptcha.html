This module contains the getters and setters for the backend classes: image/string generator, reversible encrypter, and Rack middleware. This allows you to replace them with your custom implementations. For more information on how to bring Validates Captcha to use your own implementation instead of the default one, consult the documentation for the specific default class.

This module also contains convenience wrapper methods for all the methods provided by the configured backend classes. These wrapper methods form the API that is visible to the outside world and that all backend classes use for internal communication.
