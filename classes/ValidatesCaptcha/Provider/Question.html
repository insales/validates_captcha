call (env)

This method is the one called by Rack.

It returns HTTP status 404 if the path is not recognized. If the path is recognized, it returns HTTP status 200 and delivers a new challenge in JSON format.

Please take a look at the source code if you want to learn more.

render_regenerate_challenge_link (sanitized_object_name, object, options = {}, html_options = {})

Returns an anchor tag that makes an AJAX request to fetch a new question and updates the captcha challenge after the request is complete.

Internally calls Rails’ link_to_remote helper method, passing the options and html_options arguments. So it relies on the Prototype javascript framework to be available on the web page.

The anchor text defaults to ‘New question’. You can set this to a custom value providing a :text key in the options hash.
